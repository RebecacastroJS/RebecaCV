## ğŸ¨ customization

### change colors

edit the css variables in `style.css`:

```css
:root {
    --primary-color: #3498db;      /* blue */
    --secondary-color: #2c3e50;    /* dark gray */
    --accent-color: #e74c3c;       /* red */
}
```

### change hero background image

in `style.css`, find the `.hero` section and replace the image url:

```css
.hero {
    background: linear-gradient(rgba(44, 62, 80, 0.8), rgba(44, 62, 80, 0.8)), 
                url('your-image.jpg');
}
```

### update projects

add your own projects by copying a card block in `index.html` and updating the image, title, description, and link.
